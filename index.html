À propos du programme

Ce projet met en pratique les compétences acquises en formation à travers des travaux collaboratifs utilisant Git, GitHub et les technologies web modernes.

Objectifs pédagogiques

  • Maîtriser le versionnement de code avec Git
  • Apprendre la collaboration via GitHub (pull requests, code reviews)
  • Développer un site web statique en HTML sémantique
  • Appliquer les bonnes pratiques d'accessibilité
  • Présenter son travail et rédiger une documentation claire
